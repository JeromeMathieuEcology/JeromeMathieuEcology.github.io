Visualisation des données Principes et applications avec R

Un module pour les doctorant.e.s de Sorbonne Universite

Objectifs

Ce module a pour objectif de présenter les aspects théoriques et pratiques pour explorer vos données et transformer vos graphs en visualisations à fort impact. Nous verrons comment choisir un type de visualisation, comment le mettre en place dans R, comment l'optimiser pour accroitre son impact, et comment atteindre un graph de qualité professionnelle directement publiable.
Différentes techniques de visualisation seront présentées, depuis les graphs de base jusqu'aux figures composites complètement personnalisées. Chaque technique sera présentée par un cours suivi d'une application sur un jeu de données.
Le but du module est de vous affranchir des canevas de graphiques pré-établis, afin de produire les figures les plus appropriés pour vos questions. L'accent est également mis sur la reproductibilité des figures.
